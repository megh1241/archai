Custom function evaluator.

This evaluator is used to wrap a custom evaluation function.

evaluate(model: archai.discrete_search.api.archai_model.ArchaiModel, dataset_provider: archai.api.dataset_provider.DatasetProvider, budget: Optional[float] = None) → float[source]#

Evaluate an ArchaiModel instance, optionally using a DatasetProvider and budget value.

Parameters

  • arch – Model to be evaluated.

  • dataset – A dataset provider object.

  • budget – A budget multiplier value, used by search algorithms like SuccessiveHalving to specify how much compute should be spent in this evaluation. In order to use this type of search algorithm, the implementation of evaluate() must use the passed budget value accordingly.

Returns

Evaluation result.

class archai.discrete_search.evaluators.onnx_model.AvgOnnxLatency(input_shape: Union[Tuple[int, ...], List[Tuple[int, ...]]], num_trials: Optional[int] = 1, input_dtype: Optional[str] = 'torch.FloatTensor', rand_range: Optional[Tuple[float, float]] = (0.0, 1.0), export_kwargs: Optional[Dict[str, Any]] = None, inf_session_kwargs: Optional[Dict[str, Any]] = None)[source]#

Ray#

class archai.discrete_search.evaluators.ray.RayParallelEvaluator(obj: archai.discrete_search.api.model_evaluator.ModelEvaluator, timeout: Optional[float] = None, force_stop: Optional[bool] = False, **ray_kwargs)[source]#

Wraps a ModelEvaluator object into an AsyncModelEvaluator with parallel execution using Ray.

RayParallelEvaluator expects a stateless objective function as input, meaning that any ModelEvaluator.evaluate(arch, …) will not alter the state of obj or arch in any way.
